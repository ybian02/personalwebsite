Using Clarity Design System to build a Sample App that helps people to use Clarity -

Clarity Web Builder

UI/UX | Research | Web Design

Medium: Sketch, Invision, Principle.io

In the summer of 2017, I worked as a UI/UX design intern on Project Clarity at VMware. I designed Clarity Web Builder, which creates web pages that follow Clarity design guidelines. In a span of eight weeks, I conducted user research, compiled competitive analysis, went through design iterations, designed and conducted user testing sessions. The project will be one of the first Sample apps for the Clarity Design System. Take a look at my presentation or reach out to me if you are interested in learning more about the project.
