Tufts Catering Website

UI/UX | Website Design

Medium: Adobe Photoshop, Illustrator, HTML, CSS

For my final project for Web Design class, I redesigned a website for my client, Tufts Catering. The old catering site had very limited space for images of food. Furthermore, the navigation system is extremely confusing. The primary navigation bar appears twice, under which there were two other levels of navigations, which made it difficult for users to find the information they needed. The new website features a new navigational system that is more clear and visually engaging.
